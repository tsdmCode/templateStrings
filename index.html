Template strings i JavaScript

Template strings er en måde at opbygge strenge på i JavaScript, hvor man kan inkludere variabler, udtryk og endda andre funktioner. Man kan bruge template strings til at gøre det nemmere at opbygge komplekse strenge uden at skulle bekymre sig om at sammenkæde dem manuelt.

I JavaScript oprettes template strings ved hjælp af backticks 

` eks. `
. Man kan inkludere variabler og udtryk i en template string ved at bruge syntaksen ${udtryk}.

Eksempel

Her er et eksempel på, hvordan man kan bruge template strings i JavaScript til at opbygge en besked: 


      const navn = "Keld";
      const alder = 30;
      const besked = `Hej, mit navn er ${navn} og jeg er ${alder} år gammel.`;
      console.log(besked);
      // Resultat: Hej, mit navn er Keld og jeg er 30 år gammel.

I dette eksempel bruges template strings til at opbygge en besked, og resultatet bliver vist i konsollen ved hjælp af console.log() funktionen. Variablerne navn og alder sættes ind i beskeden ved hjælp af ${}-syntaksen.
